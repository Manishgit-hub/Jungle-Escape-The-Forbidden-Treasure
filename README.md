# Jungle Escape: The Forbidden Treasure

## Overview

Jungle Escape: The Forbidden Treasure is a 3D adventure-survival game developed using Unity and C#.

Players must navigate through a dangerous jungle environment, avoid enemies, collect rewards, and reach the hidden treasure while managing their health and survival.

## Features

* Third-person player movement
* Enemy AI and attack mechanics
* Health and damage system
* Coin collection system
* Treasure-based win condition
* Game over and victory screens
* Jungle terrain environment
* Mixamo character animations
* Interactive gameplay UI

## Technologies Used

* Unity Engine
* C#
* Blender
* Mixamo Animations

## Gameplay

The player explores a jungle environment while avoiding enemies and collecting resources. The objective is to survive the journey and reach the forbidden treasure.
